How to Use

1. Name the image file in English or Numbers. (e.g. julia.jpeg)

2. Choose a file type between Image and Video.

3. Upload the image file you want to convert.

4. Press the Run button and wait for the result.

*It usually takes 10 seconds for an image file, 20 seconds for a video file.

*If you insert a video, it will be converted by cutting only the first 10 seconds of the video.
